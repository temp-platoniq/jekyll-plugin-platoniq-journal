Jekyll Plugin Platoniq Journal
==============================

A collection of plugins for Jekyll:

- Author Tag
- Call To Action Block
- Documents Tag
- File Tag
- Gallery Tag
- Links Tag
- Numbers Tag
- Questions Tag
- Quote Block
- Stickers Tag
- Video Tag
- Image Tag
- Metrics Tag

## Installation

Add this line to your application's Gemfile:

    gem 'jekyll-plugin-platoniq-journal'

And then execute:

    $ bundle

## Usage

Tags can be used in a `.md` file
